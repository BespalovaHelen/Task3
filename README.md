# Проект: Прокат музыкальных инструментов для концертов

## Описание программы

Программа демонстрирует иерархию классов с **ромбовидным наследованием** (diamond inheritance) на языке C++.

**Тема:** Прокат музыкальных инструментов для концертов.

Пользователь может:
- Сдавать инструменты в прокат (струнные, электронные, электроскрипки)
- Просматривать список доступных инструментов
- Получать описание звучания инструмента
- Возвращать инструменты из проката
- Запускать тест полиморфизма
